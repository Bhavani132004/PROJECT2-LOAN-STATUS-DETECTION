# Loan Status Prediction using Machine Learning

## Project Overview

This project predicts whether a loan application will be Approved or Rejected using Machine Learning techniques.

The model preprocesses applicant information, encodes categorical features, trains a Support Vector Machine (SVM) classifier, and predicts loan approval status.

---

## Features

- Data Cleaning
- Missing Value Handling
- Label Encoding
- Data Visualization
- Feature Selection
- Train-Test Split
- SVM Classification
- Accuracy Evaluation
- Loan Approval Prediction

---

## Dataset Features

- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area
- Loan Status

Target Variable:

- 1 → Approved
- 0 → Rejected
